Trail Making Test

Vous arrivez sur une page de test gérée par le Service de Psychologie Cognitive et de Neuropsychologie de l'Université de Mons.
Afin d'assurer votre anonymat mais de pouvoir analyser l'ensemble de vos résultats à tous les tests, nous vous demandons d'entrer un code de participant.
Votre Code participant se compose de
Les deux premières lettres de votre nom de famille
Les deux premières lettre de votre prénom
Le Chiffre (ou nombre de votre jour de naissance)
Exemple :
Sonia
Tassignon
Née le 05 novembre 1984
Code : SOTA05


ENTRAÎNEMENT PARTIE A

Bienvenue dans le Trail Making Test.

Commençons par la phase A du test, dans laquelle vous devez relier des points dans l'ordre croissant.

Pour réaliser ce test, vous allez commencer par un petit entraînement.

Vous devez cliquer sur les nombres dans l'ordre croissant, en allant le plus vite possible et en essayant de ne pas faire d'erreurs.
Phase: Temps: 0.0s Erreurs: 0

TEST PARTIE A

Vous avez terminé l'entraînement. Vous allez maintenant réaliser la phase de test dont les résultats seront enregistrés.

Comme lors de l'entraînement, vous devez cliquer sur les nombres dans l'ordre croissant, en allant le plus vite possible et en essayant de ne pas faire d'erreurs.

ENTRAÎNEMENT PARTIE B

Bravo, vous avez terminé la partie A de ce test! Vous allez maintenant réaliser la partie B.

Ici la règle change.

Les cercles à l'écran contiennent soit un nombre (1, 2, 3, ...) soit une lettre (A, B, C, ...).

En commençant par le 1, vous devez alterner les chiffres et les lettres et suivre l'ordre croissant des chiffres et l'ordre alphabétique des lettres (1 - A - 2 - B - 3 - C ...).

Répondez le plus vite possible et en essayant de ne pas faire d'erreurs.

TEST PARTIE B

Vous avez terminé l'entraînement de cette partie B. Vous allez maintenant réaliser la phase de test dont les résultats seront enregistrés.

Pour rappel, les cercles à l'écran contiennent soit un nombre (1, 2, 3, ...) soit une lettre (A, B, C, ...).

En commençant par le 1, vous devez alterner les chiffres et les lettres et suivre l'ordre croissant des chiffres et l'ordre alphabétique des lettres (1 - A - 2 - B - 3 - C ...).

Répondez le plus vite possible et en essayant de ne pas faire d'erreurs.

Test terminé !

Merci d'avoir participé au Trail Making Test !
